NC AI — AI Written in NC

The NC AI model itself is written in NC — not C, not Python. NC all the way down.

NC Transformer

Written in NCDecoder-only transformer

Model architecture defined in NC. Engine handles math.

NC Tokenizer

Written in NCBPE tokenization

NC-specific vocabulary. Encodes/decodes NC source code.

NC Optimizer

Written in NCAdam optimizer

Training loop, gradient clipping, LR scheduling — all NC.

Architecture Philosophy

Engine (C) handles: VM, compiler, tensor math — like CPython handles Python.

NC handles: model logic, training, inference, APIs, tests, scripts — everything else.

Just like PyTorch: Python defines the model, C++ handles the math. NC defines the model, the C engine handles the math.

Get Started in 60 Seconds

Install NC and write your first AI service immediately

1. Install NC

  • macOS: curl -fsSL install.nc.devheallabs.in | bash
  • Windows: Download nc.exe from releases
  • Linux: curl -fsSL install.nc.devheallabs.in | bash

2. Write your service

  • Create hello.nc
  • service "hello" / version "1.0.0"
  • to greet:
  • respond with "Hello NC!"
  • api: GET /hello runs greet

3. Run it

  • nc serve hello.nc
  • Open http://localhost:8000/hello
  • See: Hello NC!
  • That's it — zero config needed
